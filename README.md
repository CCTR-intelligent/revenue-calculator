# 共享充电收益测算页

这是一个可直接部署到 GitHub Pages 的静态网页项目。页面交互由浏览器端 JavaScript 完成，不需要服务器或构建工具。

## 文件说明

- `index.html`: 网页入口文件，包含样式、内容和交互脚本。
- `.nojekyll`: 让 GitHub Pages 按普通静态文件发布。

## GitHub Pages 发布

1. 新建一个 GitHub 仓库。
2. 上传本文件夹内的全部文件到仓库根目录。
3. 在仓库设置中打开 `Pages`。
4. Source 选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub 生成公开访问链接。

页面中的滑块测算、滚动动画和按钮跳转都会在 GitHub Pages 公网链接中继续运行。
